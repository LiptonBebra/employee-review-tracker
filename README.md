# Employee Review Tracker

Веб-приложение для хранения результатов периодической оценки сотрудников.

## Функциональность

- Регистрация и авторизация пользователей
- Три роли: Сотрудник, Менеджер, Администратор
- Просмотр оценок для сотрудников
- Управление оценками для руководителей
- Фильтрация и поиск по дате/имени
- CRUD операции для оценок
- Аккуратный интерфейс с Tailwind CSS

## Установка и запуск

### 1. Требования
- PHP 7.4 или выше
- MySQL 5.7 или выше
- Composer
- Веб-сервер (Apache/Nginx)
